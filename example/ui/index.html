jQuery Unand

What is jQuery Unand?

Unand jQuery is a JavaScript and CSS framework designed to make web-based applications on mobile devices. jQuery Unand are designed to simplify the program code created by programmers. jQuery Unand aims to minimize the resources used and the size of the file to be loaded by the browser so it can run on devices with lower specs.

What is the Anvantages?

The advantages of using jQuery Unand are as follows:

  1. Llibrary file size and file a small program that saves Internet bandwidth usage, time of access, storage, and RAM.
  2. The number of threads is small so it makes the work for the processor and RAM.
  3. Less JavaScript code usage.
  4. Resources that can be selected as appropriate.
  5. The use of vector icon format TTF, WOFF, EOT, and SVG that is compatible with virtually all modern browsers so the quality is not decreased if the page icon is enlarged, has a very small size compared with raster icons, and colors can be changed by using the CSS code as plain text.
  6. Open source, so that the framework can be freely modified as needed.
