What is jQuery Unand?

jQuery Unand is a JavaScript and CSS framework designed to make web-based applications on mobile devices. jQuery Unand is designed to simplify source code created by programmers. jQuery Unand aims to minimize the resources used and the size of the file to be loaded by the browser so it can run on devices with lower specs.

What is the Anvantages?

The advantages of using jQuery Unand are as follows:

  1. The size of program files and library files are very small so it can save internet bandwidth usage, time of transfer, RAM and storage.
  2. The number of threads is small so it makes the work for the processor and RAM.
  3. Less JavaScript code usage.
  4. Less CSS code usage.
  5. Resources that can be selected as appropriate.
  6. The use of vector icon format TTF, OTF, WOFF, EOT, and SVG that is compatible with virtually all modern browsers so the quality is not decreased if the page icon is enlarged, has a very small size compared with raster icons, and colors can be changed by using the CSS code as plain text.
  7. Open source, so that the framework can be freely modified as needed.
