Element Relationship

  1. aside
    Link to show aside
    Exp: <a href=”#aside_section” data-rel=”aside”>Show Aside</a>
  2. menu
    Link to show menu
    Exp: <a href=”#menu_section” data-rel=”menu”>Show Menu</a>
  3. popup
    Link to show popup
    Exp: <a href=”#popup_section” data-rel=”popup” data-title="Pop Up Title" data-close-icon="true" data-location-async="tools/ajax-form-login.php" data-timeout="5000" data-cache="false">Show Pop Up</a>
  4. popup-menu
    Link to show popup-menu
    Exp: <a href=”#popup_menu_section” data-rel=”popup-menu” data-target="siblings">Show Pop Up Menu</a>
  5. scroll-up
    Link to scroll up the document
    Exp: <a href=”#” data-rel=”scroll-up”>Scroll Up Document</a>
  6. scroll-down
    Link to scroll down the document
    Exp: <a href=”#” data-rel=”scroll-down”>Scroll Down Document</a>
  7. back
    Back to previous page
    Exp: <a href=”#” data-rel=”back”>Backward</a>
  8. forward
    Back to next page after back
    Exp: <a href=”#” data-rel=”forward”>Forward</a>
  9. check-all
    Check or uncheck all check box
    Exp: <input data-rel="check-all" data-target-selector=".chk1" type="checkbox" name="check-all" value="1" />
  10. dynamic-select
    Update select option using AJAX and trigger
    Exp: <select name="country" data-rel="dynamic-select" data-target-selector="#province" data-source="prov.php" data-cache="false" data-loading-caption="Wait..." data-empty-caption="Select Province"><option value="">Select Country</option><option value="ID">Indonesia</option></select>
  11. swipe-page
    Select specified page from swipe page using navigation, e.g. using navbar
    Exp 1 (using section): <nav data-role="navbar" data-rel="swipe-page" href="#swipe-page-1"><a href="#">1</a> <a href="#">2</a> <a href="#">3</a></nav>
    Exp 2 (using selector): <nav data-role="navbar" data-rel="swipe-page" data-target-selector="#swipe-page-1"><a href="#">1</a> <a href="#">2</a> <a href="#">3</a></nav>
